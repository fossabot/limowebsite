Limo Server Apt Repository

Wir freuen uns, Ihnen mitteilen zu können, dass der Limo-Server jetzt über ein eigenes Apt-Repository verfügt.

Sie können jetzt die neuesten und besten Pakete direkt von unserem Repository installieren.

Um die Repository zu nutzen, fügen Sie bitte die folgende Zeile zu Ihrer /etc/apt/sources.list Datei hinzu:

deb http://192.168.178.121/DONOTCOPY/my-apt-repo/ bookworm main

Beachten Sie, dass "bookworm" der Codename Ihrer Debian-Version sein könnte. Falls ihr System auf Bullseye basiert haben wir dafür auch einen Bereich.

deb http://192.168.178.121/DONOTCOPY/my-apt-repo/ bullseye main Zum Apt Repository
