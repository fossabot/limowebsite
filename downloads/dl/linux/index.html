Installation Guide: Linux/ARM64

Requirements:

1. You need Pythons manager Pip or Pip3

2. You need at least Python 3.7 or higher

3. You need an Internet Connection

4. You need an Account for OneDev, or an tool to Download the Source Code from the Site

5. If you have an OneDev account on "/limowebsite/" you'll need access to this repo

 

Guide:

Just run Pip:

pip install --extra-index-url https://<OneDev_account_name>:<OneDev_password>@/limowebsite//limo-games/~pypi/simple/ limo-games

 

Replace <OneDev_account_name> and <OneDev_password> with your OneDev Username and Password
